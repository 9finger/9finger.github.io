fofa介绍

FOFA是白帽汇推出的一款网络空间资产搜索引擎。
它能够帮助用户迅速进行网络资产匹配、加快后续工作进程。
例如进行漏洞影响范围分析、应用分布统计、应用流行度排名统计等。
